## Designed for Users

Native는 플랫폼의 특성을 가장 잘 활용하는 사용자 경험을 설계합니다.

iOS는 iOS답게,
Android는 Android답게.

각 플랫폼의 디자인 언어와 인터랙션을 존중하며,
사용자가 별도의 학습 없이도 익숙하게 사용할 수 있는 경험을 제공합니다.

우리는 플랫폼에 맞게 구현하고,
사용자에게는 자연스럽게 전달되는 제품을 만듭니다.

> **Designed for the platform. Built for people.**
> 사용자가 배우지 않아도 자연스럽게 사용할 수 있는 앱을 만듭니다.
